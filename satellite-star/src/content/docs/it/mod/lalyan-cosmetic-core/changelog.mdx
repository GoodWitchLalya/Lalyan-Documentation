##### <span style="color:#b96ad9" id="note_1">Nota 1</span>
Ora devi mettere la prima lettera di una chiave con l'iniziale maiuscola.<br/>
Esempio:

vecchio:
> "texture": "link"

nuovo:
> "Texture": "link"

Ulteriori informazioni [qui](/Lalyan-Projects/it/mod/lalyan-cosmetic-core/documentation/#cosmetici-avanzati-opzionale), Così è come funziona adesso.

##### <span style="color:#b96ad9" id="note_2">Nota 2</span>
La configurazione per il valore **Permissions** funziona così:

Se il valore è **true**: <br/>
Il sistema di permessi entrerà in gioco permettendo di vedere ed usare un dato cosmetico solamente se si ha il permesso necessario.

Se il valore è **false**: <br/>
L'API funziona come ha sempre fatto: puoi usare tutto.

Il valore di base è impostato su **false**.

Ulteriori informazioni [qui](/Lalyan-Projects/it/mod/lalyan-cosmetic-core/documentation/#permessi-opzionale).
